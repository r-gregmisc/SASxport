exportGraphToPowerPoint(MSOfficeUtil)R Documentation

Export Graph(s) to MS-PowerPoint

Description

Export one or more lattice graph(s) to MS-PowerPoint.

Usage

exportGraphToPowerPoint(object, file.name, overwrite=TRUE, ...)

Arguments

object A list of lattice graphs.
file.name String with full path and PowerPoint file name.
overwrite If the file exists and overwrite is FALSE, will append graphs to the end of the PowerPoint document. Otherwise, it will erase the file (or create it).
... Optional arguments

Author(s)

Steven Novick <steven.j.novick@gsk.com>

Examples

  p1 = xyplot(rnorm(10)~1:10, main="Plot 1")
  p2 = xyplot(rnorm(10)~(-10:(-1)), col="red", type="l",
        main="Plot 2", xlab="Hi there")

  exportGraphToPowerPoint(list(p1, p2), "C:/MyTest1.ppt")
  
  lset()  ## For a white background.
  exportGraphToPowerPoint(list(p1, p2), "C:/MyTest2.ppt")
