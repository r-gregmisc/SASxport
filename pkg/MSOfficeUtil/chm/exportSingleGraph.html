exportSingleGraph(MSOfficeUtil)R Documentation

MSOfficeUtil utility functions

Description

Not meant to be called directly by user.

Usage

  exportSingleGraphToPowerPoint(p, slides, theme, ...)
  exportSingleGraphToWord(p, theme, ...)

Arguments

p An object of class "lattice".
slides The Visual Basic variable containing the slides of the open PowerPoint document.
theme Lattice theme. See lset() or lattice.par.get() for details.
... Optional arguments
