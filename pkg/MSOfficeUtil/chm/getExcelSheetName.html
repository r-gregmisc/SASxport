getExcelSheetNames(MSOfficeUtil)R Documentation

Get MS-Excel sheet names

Description

Get the names of the worksheets in a MS-Excel file.

Usage

getExcelSheetNames(file.name)

Arguments

file.name String with full path and Excel file name.

Value

Returns a String vector.

Author(s)

Steven Novick <steven.j.novick@gsk.com>

Examples


  # Use example MS-Excel File included with the package
  file.name <- file.path(.path.package("MSOfficeUtil"), "Examples",
                         "Grades.xls")
  # ~= C:\Program Files\R\rwNNNN\library\MSOfficeUtil\Examples\Grades.xls

  getExcelSheetNames(file.name)
